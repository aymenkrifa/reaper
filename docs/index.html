reaper

a simple Linux TUI for listing and reaping listening ports

01 · the tui

~$ reaper 4 listening PID COMMAND PORT AGE ▸ 48213 node :3000 3d reap? [y/n] y 512 postgres :5432 12d 9021 python :8000 4h ✓ :3000 reaped — 3 still listening

Search with /, sort with s or 1–7, reap with Enter — always with confirmation.

02 · install

# with cargo (Rust 1.85+) ~$ cargo install --git https://github.com/aymenkrifa/reaper --locked

Linux only — reaper reads /proc directly. Run with sudo to see and reap other users' processes.
